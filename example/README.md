# plump_icons_example

Example app for [plump_icons](../README.md) — a searchable browser for 500 Plump icons (Line + Solid).

## Run

```bash
cd example
flutter pub get
flutter run
```

Web:

```bash
flutter run -d chrome
```

Or use the **Example (web)** launch configuration in `.vscode/launch.json`.

## What it demonstrates

- Search icons by name
- Filter by **All**, **Line**, or **Solid**
- Tap an icon to preview both styles and copy `PlumpIcon` / `BottomNavigationBarItem` snippets

Package usage and API details are in the [root README](../README.md).
